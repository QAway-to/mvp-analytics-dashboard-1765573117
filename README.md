# Analytics Dashboard MVP

Демо-дэшборд с аналитикой для «продающего» MVP.

## Возможности
- KPI карточки с ключевыми метриками
- Графики роста и активности (Recharts)
- Лента последних событий
- CTA блоки для конверсии

## Быстрый старт
```bash
npm install
npm run dev
```

## Что доработать под клиента
- Подключить живые данные (REST / GraphQL / БД)
- Настроить роли и систему доступа
- Добавить персональные отчёты и экспорт
- Интегрировать с BI (Metabase, PowerBI, Superset)
